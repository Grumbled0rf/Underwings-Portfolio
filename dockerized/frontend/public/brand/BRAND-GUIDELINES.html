Usage Guidelines

Website

  • Use SVG favicons for crisp display
  • Include tagline on desktop views
  • Hide tagline on mobile for space
  • Maintain consistent spacing

Documents

  • Use horizontal logo with tagline
  • Minimum logo height: 24px
  • Clear space: 1x logo height
  • Prefer dark version on white paper

Social Media

  • Use provided OG image template
  • Icon-only for profile pictures
  • Include gradient accent elements
  • Maintain brand color consistency

Email Signatures

  • Use horizontal logo (max height 40px)
  • Link to website
  • Include tagline when space permits
  • Use PNG fallback for compatibility
